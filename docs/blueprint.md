# **App Name**: Nano Fish Control

## Core Features:

- Live Video Stream: Display live video feed from the Nano Fish robot's camera with low latency.
- Telemetry Data Overlay: Overlay telemetry data (depth, compass, battery) on the video feed.
- Sensor Data Dashboard: Display sensor data (temperature, pH, pressure, connection) in a glass UI sidebar.
- Virtual Joystick Control: Control the Nano Fish robot's movement with a virtual joystick and buttons.
- Mission Control: Implement function buttons such as start/stop recording, take a picture and return to surface.
- Alert System: Display real-time alerts and logs for critical information (low battery, connection status).

## Style Guidelines:

- Background color: Very dark blue (#0B0F1A) to emphasize data and create a high-tech feel.
- Primary color: Neon blue (#00C8FF) for a vibrant, informative highlight.
- Accent color: Neon green (#5CFF7B) to accentuate key metrics and controls.
- Font: 'Inter', a grotesque-style sans-serif for clear readability and a modern aesthetic.
- Use thin line icons for a professional and modern feel.
- Main viewport with live video feed taking up 60-70% of the screen with an overlay of translucent data readouts.
- Glass UI sidebar dashboard for telemetry data and controls. Layout will contain alerts and logs displayed in a mini bar, located in the lower right corner.
- Implement smooth transitions for data updates and a shake animation for warnings.
